# cstat

Abstract

This study consists in predicting house prices with two different methods: Classical statistical inference methods and Neural networks. 
The combined use of various statistical tools (descriptive statistics, ANOVA, Factorial design, Parametric regression) yielded satisfactory predictions, resulting in an RMSE of 0.17.
A relatively small Neural Netwok, on the other hand, resulted in a set of predictions with an RMSE reliably around 0.12/0.15 and didn't need any pre-analysis activity other than the preprocessing. 
In particular, with classical statistical methods, a deep analysis of each feature is needed in order to build a performing regression model. So, while this is possible with a relatively simple data set, it would become utterly expensive if not impossible to analyze data sets with more features. 
The Neural Network proved to be very consistent and was able to get good performances even when less features were given during the training. 
In conclusion, both methods are valid overall but the neural network yields more precise and reliable results with less human involvement.
